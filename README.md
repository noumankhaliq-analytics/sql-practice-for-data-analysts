# SQL Practice for Data Analysts

This repository contains structured SQL practice files covering fundamental SQL concepts used in **Data Analysis and Database Management**.

These SQL queries were part of my **SQL learning journey between 2022 and 2023**, where I practiced core database concepts and query writing using MySQL.

The SQL queries are organized into separate text files based on topic to make learning easier and more structured.


---

## 📄 File Descriptions

### 1️⃣ DDL.txt
This file contains **Data Definition Language (DDL)** commands used to define and manage database structures.

Topics covered:

- CREATE DATABASE
- DROP DATABASE
- CREATE TABLE
- SHOW DATABASES
- SHOW TABLES
- Table structure creation
- Basic table schema definitions

---

### 2️⃣ Alter_commands.txt
This file demonstrates how to modify existing database tables.

Topics covered:

- ALTER TABLE
- ADD COLUMN
- MODIFY COLUMN
- DROP COLUMN
- RENAME COLUMN
- Adding constraints
- Removing constraints

---

### 3️⃣ Retrieving_Queries.txt
This file focuses on retrieving and analyzing data from tables using SQL queries.

Topics covered:

- SELECT statements
- WHERE clause
- ORDER BY
- LIMIT
- Filtering data
- Conditional operators
- Logical operators
- Pattern matching (LIKE)
- NULL handling
- BETWEEN operator
- GROUP BY
- Aggregate functions

---

### 4️⃣ SQL Commands.txt
This file includes general SQL operations used for working with data.

Topics covered:

- INSERT statements
- UPDATE statements
- DELETE statements
- AUTO_INCREMENT usage
- Sorting data
- Data filtering techniques
